# Anker AI-NPD Copilot 结构化摘要

## 项目一句话

Anker AI-NPD Copilot 是一套面向安克消费电子新品定义的证据链驱动 NPD 决策系统，帮助企业从用户反馈、竞品缺口和多智能体评审中生成可追溯的新品定义与 PRD 摘要。

## 不是做什么

本方案不是普通 AI 聊天机器人，也不是单次生成新品创意的工具。

## 是做什么

本方案通过 Evidence-to-Decision 方法论，将分散数据转化为候选新品机会、Stage-Gate 评分、多 Agent 评审结论、样例 NP PRD 和飞书协同资产。

## 核心模块

1. 项目工作台
2. 证据池概览
3. 用户痛点雷达
4. 竞品机会矩阵
5. Stage-Gate 决策门禁
6. 多智能体产品委员会
7. 样例 NP PRD 输出
8. 飞书协同沉淀

## 差异化

- 不是 Prompt-to-Idea，而是 Evidence-to-Decision。
- 不是多 Agent 角色扮演，而是结构化新品评审机制。
- 不是孤立网页 Demo，而是可进入飞书的企业 NPD 协同流程。
- 不是只输出一个概念产品，而是输出可追溯的 PRD 摘要和验证计划。
