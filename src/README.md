# src/

**English:** Frontend source for the Vite + React + TypeScript demo. Business content must come from `src/data/*.json`; `src/types/index.ts` is the executable data contract. Sites should preserve the seven-route structure and the frozen decision state in `docs/SITES_HANDOFF.md`.

**中文：** Vite + React + TypeScript Demo 的前端源码目录。业务内容必须来自 `src/data/*.json`，`src/types/index.ts` 是可执行的数据契约。Sites 实现时必须保留七页路由，并遵守 `docs/SITES_HANDOFF.md` 中冻结的决策状态。
